Key Features:



Trend Analysis: A detailed 10-year overview enabling a dynamic comparison of performance over time.



Comparative Visuals: Using clustered column charts to visualize the distribution of results across different years and categories.



Data Aggregation: Clear representation of "Grand Totals" to provide an immediate executive summary of the overall performance.



Tools used:

Microsoft Excel

Pivot Tables

Pivot Charts

Data Visualization



The dashboard supports interactive analysis through slicers, making it easy to explore data.
